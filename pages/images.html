there are 4 main types of images used on the web: .jpg, .gif, .png and .svg

jpg images are designed for handling large color palettes without exorbitantly increasing file size

this type of image does not support transparent pixels

gifs are designed for simple animations with limited color palettes

gifs do support transparent pixels but only as a binary option

pngs are great for anything that is not a photo or animated

pngs allow for opaque pixels making them great for logo designs or icons

svg is a vector-based graphics format, meaning it can scale up or down to any dimension without loss of quality

svgs have all the pros of pngs and should be used wherever possible
